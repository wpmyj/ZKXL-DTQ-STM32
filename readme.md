# ChangeLog:
*****************************************************************************************
* **版　　本：V0.1.0**
* **开发目的** ：此版本为自由协议的初始版本，主要是要解决在天喻协议版本上
　　　　　发现的不足.
* **特性描述** : 
	* 在串口上产看白名单不在分包发送，改大协议包场支持较长的包。
	* 增加协议的版本设备识别功能，能自适应答题器设备，采用不同的协议进行通讯。
	* 优化设置指令的分配，相关功能分配为一组。
	* 增加通讯信道参数发送功率等参数的配置接口。
